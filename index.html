Pipeline notes (all 3 examples)

Predictor
Molmo2 (rollout5: 5 sequential 6-frame chunks → T_future=30 in pred_raw_combined)
Frame mapping
stride 1: pred_raw_combined[i, t, :] = source frame t0 + 1 + t (verified via gt_future_raw exact-match)
Visibility
all-True (no per-frame vis info in prediction)
Adapter
v3 INTERP path (T<49) with frozen first-frame camera at source frame t0+1
DaS
demo.py @ 50 inference steps; new "fairly static" suffix; no --repaint
Sora
sora-2, size by GT log-aspect, seconds=4 (smallest ≥ D_target); cv2-probed actual = 30 fps; trim+resample+resize to GT res
Metrics — frame-level
PSNR/SSIM/LPIPS computed after upsampling DaS to GT res with cv2.INTER_AREA; t=0 skipped (input frame is identical across panels)
Metrics — VBench
subject_consistency / motion_smoothness / dynamic_degree via vbench evaluate --mode custom_input on the upsampled mp4s
